## Features ‚úÖ

- **CPU Monitoring**
  - Global CPU usage (%)
  - Per-core usage breakdown
- **Memory Monitoring**
  - Total, used, free, and available memory (in GB)
  - Memory usage percentage
- **Simple CLI Output**
  - Refreshes every second
- **Modular & Extensible**
  - Separate APIs for CPU and memory monitoring
